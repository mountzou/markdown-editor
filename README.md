# Markdown Editor

![Vite](https://img.shields.io/badge/Vite-8.0.0-orange) ![React](https://img.shields.io/badge/React-19.2.4-yellow) ![TypeScript](https://img.shields.io/badge/TypeScript-5.9.3-3178C6)

A client-side Markdown editor built with Vite, React, and TypeScript.

## Features

- Supports real-time Markdown preview
- Supports syntax highlighting for code blocks
- Supports LaTeX for mathematical expressions

## Key Libraries

- `@uiw/react-codemirror` 4.25.8
- `marked` 17.0.4
- `highlight.js` 11.11.1
- `katex` 0.16.38
- `turndown` 7.2.2

## Setup

```bash
npm install
npm run dev
```
